# siffer
Simple Caesar Cipher utility tools

## Install
```
nimble install https://github.com/TheBoringDude/siffer.git
```

## Usage
### Available `procs`
- `encodeRot13`
  ```nim
  import siffer

  echo encodeRot13("hello WORLD")
  ```
- `decodeRot13`
  ```nim
  import siffer

  echo decodeRot13("uryyb JBEYQ")
  ```
